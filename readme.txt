This is not just a plugin, it symbolizes the enthusiasm gaucho developers have for this awesome CMS platform and for their homeplace, as well. When activated you will randomly see a lyric from <cite>Gaud&eacute;rio</cite>, a poem by Jo&atilde;o da Cunha Vargas, in the upper right of your admin screen on every page.

== Description ==

Baseado no simples e belo "Hello, Dolly", de Matt Mullenweg, criador do Wordpress, este não é apenas um plug-in. Ele simboliza todo o entusiasmo que os desenvolvedores gaúchos têm por esta incrível plataforma e pelo Rio Grande do Sul, seu rincão. O plug-in mostra um verso do poema “Gaudério”, de João da Cunha Vargas, a cada página da tela de administração do Wordpress. 

(Based on "Hello, Dolly", by Matt Mullenweg, this plug-in shows a line from "Gaudério", a poem by João da Cunha Vargas.)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. <i>T&acute; pronto, tch&ecirc;!
